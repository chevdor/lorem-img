Here are a few examples of the `lorem-img`

This web component uses Polymer. You can find the project page for this component on GitHub: https://github.com/chevdor/lorem-img

This component is loading images from http://lorempixel.com/.

Basic usage:

You get some random image with the default size. Whatever you get is cached in your page so any other call will bring the same image unless you use the nocache attribute.

<lorem-img></lorem-img>

Specifying the size:

If you omit one dimension, it defaults to 100.

<lorem-img w="200" h="200"></lorem-img>

Specifying some text:

Watch out, if you donÂ´t provide a category, you get a cat!

<lorem-img txt="Soooooo cuuuuute"></lorem-img>

Specifying a category:

You can select one of the following categories.

<lorem-img cat="people"></lorem-img>
abstract animals business cats
city food nightlife fashion
people nature sports technics
transport

Specific image in the set.

Always get image N of the set (reload to see: it remains the same image)

<lorem-img n="1" cat="cats"></lorem-img>

No Cache

<lorem-img nocache></lorem-img>

Always ensure we get a different picture, although we use the same size and params.
